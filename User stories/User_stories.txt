### User stories

1. As a gym owner, I want to implement a points system that rewards you every time you make a new membership, so that more people will remain subscribed to my gym.

2. As a professional athlete, I want to be able to mark weights, sets and repetitions done in the gym, so that I can manage my progress.

3. As a gym owner, I want to create a referral program to incentivize my current members to bring in new clients, so that i can increase my costumers number.

4. As a personal trainer, I want to create new workout routines for my customers, so that I can personally manage their trainings.

5. As a athlete, I want to be able to ask a workout routine, so that i can train with it.

6. As a personal trainer, i want to to be able to add notes to workout routines, so that i can comunicate with my client.



Extra: 

1. As a customer, I want to see my old workout routines, so that I can compare them.

4. As a beginner customer, I want to skip sets and exercises, so that I can finish my workout more easily.

8. As a professional user, I want to see my training data (weight progess, frequency of training, weight lost, muscle gaines) in my personal page, so that i can track how my training is going.
